# JRSEC — Job Recruitment System 🧑‍💼
> A Java GUI-Based Solution for Start-Ups  

## 📌 About
A full-featured desktop recruitment platform built with Java Swing and JDBC.
Supports 4 user roles: Job Seeker, Recruiter, Admin, and Guest.

## 👤 My Role — Group Leader
- 🗂️ **System Flowchart** — designed the full process flow (login → job posting → search → application tracking)
- 🧩 **Class Diagram (UML)** — modelled system architecture with main classes: Applicant, Recruiter, Job, Application
- 🎨 **UI Design (Figma)** — created interface mockups and blueprints for the team to implement
- ⚙️ **OOP Implementation** — applied all 4 pillars: Inheritance, Encapsulation, Polymorphism, Abstraction

## 🛠️ Tech Stack
Java · Java Swing · JDBC · MySQL · Figma · UML
